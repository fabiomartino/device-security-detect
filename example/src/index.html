Capacitor Device Security Detect

@capacitor-community/device-security-detect

This plugin helps detect if a device is rooted/jailbroken, has a secure lock screen enabled, is running in an emulator, or is in debug mode. Useful for enhancing security in Capacitor apps.

Root/Jailbreak Detection: Is the device compromised? Checking... PIN/Password Check: Is secure authentication enabled? Checking... Emulator Detection: Is the app running in an emulator? Checking... Debug Mode Detection: Is the app running in debug mode? Checking...
